<summary><b>🐍 Snake Animation Setup</b></summary>

To enable the snake animation on your profile, create this file in your profile repo:

**`.github/workflows/snake.yml`**

```yaml
name: Generate Snake Animation

on:
  schedule:
    - cron: "0 */12 * * *"  # Runs every 12 hours
  workflow_dispatch:

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk@v3
        with:
          github_user_name: HemangDubey
          outputs: |
            dist/github-snake.svg
```
